Project detail

Online Cinema Booking Web Application is a customer project. A student from TU requested me to develop a web application about cinema. The application is to use in her thesis. So, I accepted this project to get more experiences. This project included three members including me. We met her to collect requirements. And then, we assigned how to develop the application. I was responsible for the main implementation of the project and uploading the application in the hosting. I used jQuery and Ajax in Frontend and PHP in Backend. In this project, I had to know about jQuery and Ajax well. The different factor between this project and my other projects is mobile view. This application can be viewed with any device.
